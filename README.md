# Hi, I'm Azmat 👋

I build platform-level systems - the kind that make other engineers faster rather than just shipping features on top of features.

At my day job I designed a config-driven framework in Node.js, TypeScript, and Vue/Nuxt that generates entire UIs, workflows, and role-based systems from a JSON config. It runs across 20+ products and cut feature delivery from weeks to a couple of hours. Most of my work lives in that layer beneath features - the abstractions, the patterns, the architecture decisions that compound over time.

Outside of that I build things that scratch my curiosity. Sometimes that's an AI document chat platform, sometimes it's implementing ping from scratch in C just to understand how it actually works at the packet level.

---

## What I'm working on

- **[Tablo](https://github.com/AzzVipe/tablo)** - open-source config-driven data management framework. Tables, forms, filters, drawers, RBAC, bulk ops - all from a JSON config, no repeated component code
- **[Briefly](https://github.com/AzzVipe/briefly)** - self-hosted RAG document chat. Upload PDFs, talk to them locally using Ollama and pgvector, nothing leaves your machine
- **[NetKit](https://github.com/AzzVipe/netkit)** - ping, traceroute, ARP reader, and passive ICMP detector written in pure C using raw sockets. No external libraries
- **[Pwdman](https://github.com/AzzVipe/pwdman)** - local password manager in C with AES-256-GCM encryption and Unix socket IPC
